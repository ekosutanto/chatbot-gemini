# 🌿 Herbal Assistant - Chatbot with Gemini AI

Sebuah chatbot berbasis AI yang membantu pengguna mendapatkan informasi tentang tanaman herbal, pengobatan tradisional, cara meracik ramuan, efek samping, dan dosis penggunaan. Dibangun menggunakan **Google Gemini AI** dan **Node.js/Express**.

![Herbal Chatbot Screenshot](https://via.placeholder.com/800x400?text=Herbal+Chatbot+Screenshot)

## ✨ Fitur

- 🤖 **Chatbot AI** - Tanya jawab tentang herbal dengan Gemini AI
- 📄 **Upload Dokumen** - Analisis dokumen herbal (PDF, DOC, TXT, Gambar)
- 🌿 **Informasi Lengkap** - Tanaman herbal, pengobatan, cara meracik, efek samping, dosis
- 💬 **Real-time Chat** - Antarmuka chat yang responsif dan interaktif
- 📱 **Mobile Friendly** - Tampilan yang optimal di semua perangkat
- 🔒 **Aman** - Menggunakan environment variables untuk API key

## 🛠️ Teknologi yang Digunakan

- **Backend**: Node.js + Express.js
- **AI**: Google Gemini AI API (gemini-1.5-flash)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **File Upload**: Multer
- **Environment**: dotenv

## 📋 Prasyarat

Sebelum menjalankan project, pastikan Anda memiliki:

- [Node.js](https://nodejs.org/) (v20 atau lebih baru)
- [npm](https://www.npmjs.com/) (terinstall bersama Node.js)
- [Google Gemini API Key](https://makersuite.google.com/app/apikey)
